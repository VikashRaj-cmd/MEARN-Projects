# 🚀 MEARN Projects

A curated collection of **full-stack applications** built using the **MEARN stack**:

> **MongoDB + Express.js + Angular / React + Node.js**

MEARN extends MERN by allowing **flexibility in frontend frameworks** — you can build projects using **React or Angular** while keeping a consistent backend architecture.

---

## 🌟 About This Repository

This repository serves as a **central hub** for all my MEARN-based projects, showcasing:

* 📦 Scalable full-stack architectures
* 🤖 AI-powered applications
* 🔐 Authentication & secure APIs
* ⚡ Modern frontend frameworks (React / Angular)
* ☁️ Cloud deployment (Vercel, Render, MongoDB Atlas)

## 🚀 Featured Project

### 🤖 Aethron: AI Learning Assistant

A full-stack **AI-powered learning assistant** designed to enhance education through intelligent interaction.

### ✨ Features

* 🧠 Real-time AI Q&A with contextual understanding
* 👤 User authentication & personalized dashboard
* 📊 Learning progress tracking
* 💬 Chat-based AI assistant
* ⚡ Fast and responsive UI

---

### 🛠️ Tech Stack

**Frontend:**

* React 18 + TypeScript
* Tailwind CSS

**Backend:**

* Node.js + Express.js

**Database:**

* MongoDB Atlas

**AI Integration:**

* OpenAI / Gemini API

### ⚙️ Getting Started

```bash
# Clone the repository
git clone https://github.com/your-username/MEARN-Projects.git

# Navigate to project
cd MEARN-Projects/aethron

# Install dependencies
npm install

# Run the project
npm run dev
```
