MEARN Stack

A curated collection of full-stack applications built with MongoDB, Express, Angular/React, and Node.js.

MEARN = MERN + Angular flexibility. Choose React or Angular per project — same backend, modern frontend patterns.

🚀 Featured Project
Aethron: AI Learning Assistant
A full-stack AI-powered learning companion for students and educators.

✅ Real-time Q&A with contextual understanding
✅ User accounts, progress tracking, and personalized dashboards
✅ Express + MongoDB backend, React + TypeScript frontend
✅ Deployed on Vercel (frontend) & Render (backend)
Tech: React 18, TypeScript, Tailwind CSS, Express, MongoDB Atlas, OpenAI API

Live Demo: https://aethron.vercel.app

Getting Started:

# Clone repo
git clone https://github.com/your-username/MEARN.git
cd MEARN/aethron

# Install & run
npm install && npm run dev

🌟 Why MEARN?
Frontend Agnostic: Build with React or Angular — same backend contracts.
Production-Ready: Auth, validation, error handling, and CI/CD examples included.
AI-Native: Designed with LLM integration, RAG, and agent patterns in mind.
Open & Learnable: Clear docs, consistent structure, MIT licensed.
